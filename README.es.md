# 📦 Things - Just Things

> **Organiza tu mente, una cosa a la vez.**

**[English](README.md)** | **[Español](README.es.md)** | **[Galego](README.gl.md)**

---

Things es un proyecto de código abierto enfocado en facilitar la organización y la tranquilidad mental de los usuarios. Te permite organizar y almacenar información de interés para recuperarla cuando realmente importa.

Creado por el **LleidaHackers Team** en **HackUDC** 🚀

---

## 🌟 ¿Qué es Things?

Things te ayuda a capturar y organizar información sin la carga mental de recordar todo. Guarda lo que importa ahora, recupéralo cuando lo necesites — simple, eficiente y sin estrés.

---

## 🏗️ Arquitectura

Things consta de tres componentes principales:

### 📱 [Aplicación Móvil](https://github.com/LLEIDAHACKERS/things-frontend)
Aplicación móvil nativa para iOS y Android para gestionar tus cosas sobre la marcha.

### 🧩 [Extensión de Navegador](https://github.com/LLEIDAHACKERS/things-extension)
Extensión de navegador para guardar rápidamente contenido de la web directamente en tu colección de Things.

### ⚙️ [Backend](https://github.com/LLEIDAHACKERS/things-backend)
Sistema de API y gestión de datos que impulsa todo el ecosistema de Things.

## 🚀 Comenzar

Este repositorio contiene los tres componentes principales como submódulos de Git. Para clonar el proyecto completo con todos los submódulos:

```bash
# Clonar con todos los submódulos
git clone --recurse-submodules https://github.com/LLEIDAHACKERS/things.git

# O si ya clonaste, inicializa los submódulos
git submodule update --init --recursive
```

Para actualizar todos los submódulos a sus últimas versiones:

```bash
git submodule update --remote --recursive
```
