# 📦 Things - Just Things

> **Organiza a túa mente, unha cousa á vez.**

**[English](README.md)** | **[Español](README.es.md)** | **[Galego](README.gl.md)**

---

Things é un proxecto de código aberto enfocado en facilitar a organización e a tranquilidade mental dos usuarios. Permíteche organizar e almacenar información de interese para recuperala cando realmente importa.

Creado polo **LleidaHackers Team** en **HackUDC** 🚀

---

## 🌟 Que é Things?

Things axúdache a capturar e organizar información sen a carga mental de recordar todo. Garda o que importa agora, recupérao cando o necesites — simple, eficiente e sen estrés.

---

## 🏗️ Arquitectura

Things consta de tres compoñentes principais:

### 📱 [Aplicación Móbil](https://github.com/LLEIDAHACKERS/things-frontend)
Aplicación móbil nativa para iOS e Android para xestionar as túas cousas sobre a marcha.

### 🧩 [Extensión de Navegador](https://github.com/LLEIDAHACKERS/things-extension)
Extensión de navegador para gardar rapidamente contido da web directamente na túa colección de Things.

### ⚙️ [Backend](https://github.com/LLEIDAHACKERS/things-backend)
Sistema de API e xestión de datos que impulsa todo o ecosistema de Things.

## 🚀 Comezar

Este repositorio contén os tres compoñentes principais como submódulos de Git. Para clonar o proxecto completo con todos os submódulos:

```bash
# Clonar con todos os submódulos
git clone --recurse-submodules https://github.com/LLEIDAHACKERS/things.git

# Ou se xa clonaches, inicializa os submódulos
git submodule update --init --recursive
```

Para actualizar todos os submódulos ás súas últimas versións:

```bash
git submodule update --remote --recursive
```
